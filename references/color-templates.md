## 模板 A：Claude 风格 - 高密度专业知识图谱（默认）

**风格：** 专业矢量图标 + 23 模块蛇形路径 + 顶部横幅 + 图例键

**配色逻辑：** Claude 品牌色 + 4 层级色彩系统

### 核心配色

| 层级 | 用途 | 色值 | 说明 |
|------|------|------|------|
| **画布背景** | Background | #F5F2ED | 暖米白/奶油色，柔和专业 |
| **顶部横幅** | Header Banner | #D97B54 | 深赤陶红/土红，纯色填充 |
| **核心模块** | Core Modules | #D97B54 | 深赤陶红，用于架构核心模块 |
| **功能模块** | Feature Modules | #F8D7BF | 柔和桃色/沙色，用于功能模块 |
| **集成模块** | Integration | #2D2926 | 深炭灰/石墨色，用于深度集成 |
| **配置模块** | Config Files | #F8D7BF + 细边框 | 浅桃色，用于配置文件 |
| **文字（深色背景）** | Text on Dark | #FFFFFF | 纯白，用于深色模块内文字 |
| **文字（浅色背景）** | Text on Light | #2D2926 | 深炭灰，用于浅色模块内文字 |
| **模块边框** | Module Border | #000000 | 粗黑描边，3-4px |
| **蛇形路径** | Snake Path | #2D2926 | 深炭灰短虚线 + 箭头，2-3px |

### 装饰元素

| 元素 | 色值 | 说明 |
|------|------|------|
| **Claude Logo 花纹** | #D97B54, 20-40% 不透明度 | 手绘风格，顶部横幅左右两侧 |
| **图例键** | 多色圆圈 | Architecture(赤陶红)/Core(桃色)/Integrations(深色)/Config(浅桃色)/External(深橙) |

### 模块样式规范

| 属性 | 值 | 说明 |
|------|-----|------|
| **形状** | 圆角矩形 | 12-16px 圆角半径 |
| **标题栏** | 圆角矩形 | 15-18px 圆角半径，小标题区域 |
| **边框** | #000000, 2-3px | 所有模块都有黑描边 |
| **填充** | 纯色 | 无渐变 |
| **间距** | 20-30px | 模块之间保持一致间隙 |
| **结构** | 23 个连续模块 | 单一、长的、连续的蛇形流 |

### 蛇形路径规范

- **虚线样式**：短虚线（不是长虚线）
- **箭头**：每个线段末端都有箭头，指向下一个模块
- **路径序列**：顶部横幅 → 模块 1→2→3→...→23 → 底部页脚
- **颜色**：深炭灰 #2D2926
- **线宽**：2-3px
- **流向**：从左到右 → 从右到左 → 再从左到右（S 形蜿蜒）

### 排版规范

| 元素 | 字体 | 样式 |
|------|------|------|
| **主标题** | Inter/PingFang Bold | 大型，白色，全大写 |
| **副标题** | Inter/PingFang Regular | 中型，白色 |
| **模块标题** | Inter/PingFang Bold | 粗体，不编号 |
| **正文** | Inter/PingFang Regular | 干净，多行，行高 1.6 |
| **页脚** | Inter/PingFang Regular | 小字体，10-12pt，深炭灰 |

### 顶部横幅规范

- **位置**：画布顶部，横跨整个宽度
- **高度**：约占画布的 12-15%
- **背景**：纯色赤陶红 #D97B54
- **标题**：白色粗体大字 "OPENCLAW MEMORY SYSTEM ARCHITECTURE"
- **副标题**：白色 "Every component, explained in technical clarity"
- **图例键栏**：横幅正下方，水平排列，彩色圆圈 + 文字

### 页脚规范

- **右下角**："skill by JunEr | Copyright 2026 OpenClaw Solutions"
- **颜色**：深炭灰 #2D2926

### 适用场景

- 系统架构图解
- 专业知识图谱
- 技术组件详解
- 产品功能全景
- AI 模型/框架介绍

### 使用方式

```
"用 Claude 风格" → 模板 A
"用专业知识图谱" → 模板 A
"用模板 A" → 模板 A
```

### 禁止事项

❌ 画布上直接写标题（必须在横幅中）
❌ 深色背景上用深色文字（用白色）
❌ 长虚线（用短虚线）
❌ 没有箭头的路径
❌ 缺少标题下的图例/键
❌ 缺少页脚细节
❌ 霓虹色/荧光色
❌ 卡通涂鸦
❌ 3D 效果或阴影

## 模板 B：手绘笔记风格（Sketchnote）

**风格：** 混乱手绘 + 马克笔 + 网格纸质感 + 有机瑕疵

**配色逻辑：** 马克笔 + 墨水混合效果

| 层级 | 用途 | 色值 | 说明 |
|------|------|------|------|
| **背景** | Background | #F5F5DC + 网格纹理 | 经典网格笔记本纸，带有机瑕疵（墨水飞溅、污渍） |
| **墨水色** | Ink & Text | #1A1A1A | 炭黑色，粗糙毡头马克笔和针管笔纹理 |
| **强调色** | Accent | #FF7000 | 活力橙色马克笔效果，用于标题背景、高亮、粗箭头 |
| **高光线** | White Highlight | #FFFFFF | 白色细线笔触，用于标题内的重叠描边效果 |
| **阴影** | Hatching | #1A1A1A (交叉排线) | 粗糙手绘交叉排线阴影 |

**视觉特征：**
- 主标题：粗黑马克笔 + 橙色背景块 + 白色细线重叠描边
- 布局：混乱模块化，重叠视觉集群
- 连接线：颤抖箭头、虚线路径、涂鸦、指向线
- 字体：极不规则手写体，大小/角度/对齐变化
- **涂鸦（MESSY GRAFFITI）：** 粗糙混乱的手绘图案混合（齿轮、火柴人、灯泡、放大镜、混乱图表、闪电）
- **阴影（SHADING）：** 粗糙快速的手绘交叉线（对角线）创造阴影，从而产生深度感
- **原创线条（ORIGINAL LINES）：** 所有笔触必须极度不规则、不均匀、自然流动，完全禁止直线

**适用场景：**
- 头脑风暴笔记
- 创意工作坊记录
- 个人学习笔记
- 非正式教程/指南
- 创意营销内容

**使用方式：**
```
"用手绘风格" → 模板 B
"用 Sketchnote" → 模板 B
"用模板 B" → 模板 B
```

**提示词应用示例：**
```markdown
【COLOR PALETTE - HAND-DRAWN SKETCHNOTE LOGIC】
- BACKGROUND: Weathered grid paper (#F5F5DC) with ink splatters and smudges
- INK: Charcoal Black (#1A1A1A) with raw felt-tip marker texture
- ACCENT: Vibrant Safety Orange (#FF7000) marker effect
- HIGHLIGHT: White fine-liner pen lines (#FFFFFF) for overlay tracing
- SHADOW: Rough cross-hatching with charcoal ink

【MAIN TITLE EFFECT】
- Thick black marker over rough orange marker background block
- Black letters feature thin, scratchy WHITE FINE-LINER lines tracing inside
- Multi-layered highlight and re-traced messy effect

【LAYOUT】
- CHAOTIC & MODULAR: Overlapping visual clusters
- Connect with wobbly arrows, dashed paths, scribbles, pointing lines
- NO straight lines, NO perfect shapes
```

**禁止事项：**
❌ 干净数字字体
❌ 完美直线或几何形状
❌ 干净 sterile 布局
❌ 机器生成的完美对齐
❌ 平整矢量图标
