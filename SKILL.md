# 高密度信息图生成器

基于 Nano Banana Pro (Gemini 3 Pro Image) 生成高密度、专业级信息图表。

**风格定位：** Claude 风格专业知识图谱 / 手绘笔记混乱风格

---

## 配色方案模板

### 模板 A：Claude 风格 - 高密度专业知识图谱（默认）

**风格：** 专业矢量图标 + 23 模块蛇形路径 + 顶部横幅 + 图例键

**核心配色：**

| 层级 | 用途 | 色值 | 说明 |
|------|------|------|------|
| **画布背景** | Background | #F5F2ED | 暖米白/奶油色 |
| **顶部横幅** | Header Banner | #D97B54 | 深赤陶红，纯色填充 |
| **核心模块** | Core Modules | #D97B54 | 深赤陶红，架构核心 |
| **功能模块** | Feature Modules | #F8D7BF | 柔和桃色，功能模块 |
| **集成模块** | Integration | #2D2926 | 深炭灰，深度集成 |
| **文字（深色背景）** | Text on Dark | #FFFFFF | 纯白 |
| **文字（浅色背景）** | Text on Light | #2D2926 | 深炭灰 |
| **模块边框** | Module Border | #000000 | 粗黑描边 3-4px |
| **蛇形路径** | Snake Path | #2D2926 | 短虚线 + 箭头 |

**装饰元素：**
- Claude Logo 花纹：#D97B54, 20-40% 不透明度，顶部横幅两侧
- 图例键栏：横幅下方，彩色圆圈 + 文字

**模块样式：**
- 形状：圆角矩形（12-16px 圆角）
- 边框：黑色 2-3px
- 结构：23 个连续模块，蛇形路径连接

**蛇形路径：**
- 短虚线，每段末端有箭头
- 路径：顶部横幅 → 模块 1→2→...→23 → 页脚
- 流向：左→右→左→右（S 形蜿蜒）

**页脚：**
- 左下："to download more infographics, go to openclaw.solutions"
- 右下："Copyright 2024 OpenClaw Solutions\nv2.1 Professional Knowledge Graph"
- 右边缘垂直："from OpenClaw AI"

**使用方式：**
- "用 Claude 风格" → 模板 A
- "用专业知识图谱" → 模板 A
- "用模板 A" → 模板 A
- 不指定 → 默认模板 A

**禁止：**
❌ 画布上直接写标题（必须在横幅中）
❌ 深色背景用深色文字
❌ 长虚线（用短虚线）
❌ 无箭头路径
❌ 缺少图例键或页脚

### 模板 B：手绘笔记风格（Sketchnote）

**配色逻辑：** 马克笔 + 墨水混合效果 + 网格纸质感

| 层级 | 用途 | 色值 | 说明 |
|------|------|------|------|
| **背景** | Background | #F5F5DC + 网格纹理 | 经典网格笔记本纸，带墨水飞溅、污渍等有机瑕疵 |
| **墨水色** | Ink & Text | #1A1A1A | 炭黑色，粗糙毡头马克笔和针管笔纹理 |
| **强调色** | Accent | #FF7000 | 活力橙色马克笔效果，用于标题背景、高亮、粗箭头 |
| **高光线** | White Highlight | #FFFFFF | 白色细线笔触，用于标题内重叠描边 |
| **阴影** | Hatching | #1A1A1A (交叉排线) | 粗糙手绘交叉排线阴影 |

**视觉特征：**
- 主标题：粗黑马克笔 + 橙色背景块 + 白色细线重叠描边
- 布局：混乱模块化，重叠视觉集群
- 连接线：颤抖箭头、虚线路径、涂鸦、指向线
- 字体：极不规则手写体，大小/角度/对齐变化
- **涂鸦（MESSY GRAFFITI）：** 粗糙混乱的手绘图案混合（齿轮、火柴人、灯泡、放大镜、混乱图表、闪电）
- **阴影（SHADING）：** 粗糙快速的手绘交叉线（对角线）创造阴影，从而产生深度感
- **原创线条（ORIGINAL LINES）：** 所有笔触必须极度不规则、不均匀、自然流动，完全禁止直线

**使用方式：**
- "用手绘风格" → 模板 B
- "用 Sketchnote" → 模板 B
- "用模板 B" → 模板 B

**禁止：** 干净数字字体、完美直线、sterile 布局、机器生成的完美对齐

## 使用流程

### 步骤 1：接收输入
**用户输入格式：**
- 链接：URL 地址（网页、文章、文档）
- 文章：直接粘贴文本内容
- 可选：指定配色方案（"用 Claude 配色"或"用模板 A"）
- 可选：指定图片数量（"生成 3 张"或"做一套图"）

**自动处理：**
1. 使用 `web_fetch` 或 `read` 工具读取链接/文章内容
2. 自动提炼核心内容、关键数据、核心观点
3. 智能判断需要生成的图片数量（1-10 张）

### 步骤 2：深度搜索（可选）
**触发条件：** 仅当以下内容不足时启用：
- 缺少价格区间、技术参数、使用寿命等关键数据
- 用户明确要求"搜索更多数据"或"补充信息"

**搜索工具：** `agent-reach` 或 `searxng`

### 步骤 3：提炼价值 & 视觉坐标拆分
**提炼标准：**
- 实用性：可直接执行的建议
- 稀缺性：深度洞察
- 清晰性：解决具体痛点

### 步骤 4：选择配色方案
**默认：** 模板 A（Claude 风格专业知识图谱）
**用户指定：**
- "用 Claude 风格"→模板 A
- "用专业知识图谱"→模板 A
- "用模板 A"→模板 A
- "用手绘风格"→模板 B
- "用 Sketchnote"→模板 B
- "用模板 B"→模板 B

### 步骤 5：生成提示词
使用 `prompts/infographic-prompt-template.txt` 模板，根据选择的配色方案调整色彩参数。

### 步骤 6：调用 Nano Banana Pro
```bash
uv run ~/.openclaw/workspace/skills/nano-banana-pro/scripts/generate_image.py \
  --prompt "[生成的提示词]" \
  --filename "YYYY-MM-DD-HH-MM-SS-[主题]-01.png" \
  --resolution 2K  # 默认 2K，可选 4K
```

## 输出规格

- **比例：** 3:4 (竖版)
- **分辨率：** 2K (默认推荐) / 4K (可选)
- **模块数：**
  - 模板 A（Claude 风格）：10-23 个连续模块（根据文章内容，最少 10 个）
  - 模板 B（手绘风格）：10-15 个模块（根据文章内容，最少 10 个）
- **页脚：** 右下角小字 "skill by JunEr | Copyright 2026 OpenClaw Solutions"
- **图片数量：** 智能判断（1-10 张），或用户指定

**分辨率建议：**
- **2K (1920x2560)** - 默认推荐，生成快速 (~60 秒)，文件适中 (5-10MB)
- **4K (3840x5120)** - 高质量需求，生成较慢 (~120 秒)，文件较大 (15-25MB)
